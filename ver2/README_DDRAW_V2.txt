使い方

　・同時に解凍されたファイル ddraw.dll を
　　天翔記のインストールされているフォルダにコピーしてください。
　　間違っても Windows の System32 フォルダの中には入れないでください。
　　アンインストは ddraw.dll を削除すればＯＫ。

要望

  ・十分なデバッグ時間がありません。環境とかも・・・ムリ。
　　なので問題があれば報告してね。
　　
要注意事項

　・Vistaの場合、エアロ環境でバックグラウンド動作させるとそのうちフリーズします。
　　理由はサッパリ分かりません。以下の対策をしてください。
　　TENSHOU.EXE(TENSHOU-A.exe) → プロパティ → 互換性タブ → 互換モードで実行する
　　にチェックを入れてください。(Win XP-SP2でいい模様)

　・ＯＳによる自動256色モード切替の対策
　　Vista/XPの場合、何故か実行ファイル名を変えると、自動認識しなくなる模様。
　　　例）TENSHOU.exe → TENSHOU-A.exe
　　理由はサッパリ分かりません。そもそもどうやって認識しているのか？？？
　　win2000の場合、そのまま実行可能です。

　・プロパティの互換性タブで、256色で実行するをチェックしている場合は外してください。

　・環境/製品バージョンの違いによって動かない可能性もあります。

  ・256色モードで実行すると、パレット処理が美しくありませんが仕様です。
　　互換の為に一応サポートしてあるだけで、同じ256色モードでも
　　天翔記本来の表示処理とは異なります。

Ver2変更点

　・Windows の全解像度＆全カラーモードに対応。
　　またプレイ中に解像度＆カラーモードの変更が可能。

　・バックグラウンドでの表示をするようになった。

　・Vistaのエアロでも動作するようになった。(要互換モード設定）
　　地デジチューナー等の表示もエアロ→スタンダードに移行しても、
　　問題なくプレイ継続＆視聴できる模様。

　・一応、ウィンドウサイズの自由拡大機能追加。暫定。
　　現時点では拡大画像が美しくありません。検討中です。
　　拡大は 上下 にウィンドウの端をドラッグした場合に、
　　それに合わせて左右拡大も適用されます。(縦横比固定)
　　その為、左右にドラッグしても無効となります。
　　また、拡大上限も設定してあります。(1600x1184ドットまで)
　　縮小は出来ないようにした為、限界までサイズを縮めると元のサイズに戻ります。
　　拡大時にムービー表示位置が元の位置のままですが、そのうち修正するかも。
　　640*480のサイズの場合は、拡大しないようになっています。

まったく修正する気のない不具合

　・256色モードで起動すると、その他の色モードに移行してもムービーは256色のまま。
　・256色モード以外で起動すると、256色モードに移行したらムービーの色が変。
