使い方

　・同時に解凍されたファイル ddraw.dll を
　　天翔記のインストールされているフォルダにコピーしてください。
　　間違っても Windows の System32 フォルダの中には入れないでください。
　　アンインストは ddraw.dll を削除すればＯＫ。

要望

  ・十分なデバッグ時間がありません。環境とかも・・・ムリ。
　　なので問題があれば報告して。
　　
要注意事項

　・ＯＳによる自動256色モード切替の対策
　　Vista/XPの場合、何故か実行ファイル名を変えると、自動認識しなくなる模様。
　　　例）TENSHOU.exe → TENSHOU-A.exe
　　理由はサッパリ分かりません。そもそもどうやって認識しているのか？？？
　　win2000の場合、そのまま実行可能です。

　・プロパティの互換性タブで、256色で実行するをチェックしている場合は外してください。

　・環境/製品バージョンの違いによって動かない可能性もあります。

Ver3変更点

　★フルカラー顔表示機能搭載＆簡易アニメ機能付き。
　　やってみました。破綻(貼られない＆ゴミでる)率もあります。
　　フルカラー顔画像は付属してません。作って。
　　詳しくは同梱のファイルで。
　　注）256色モードでは元の顔表示になります。
　　注）評定の画面中の顔は未対応です。(検討中)

　・かつてご指摘のございました天翔記側のオプション設定(メニューバー)の
　　「画面描画の特殊効果を使用」に対応しました。

　・Vistaエアロ環境で互換モード設定しないままバックグラウンド動作させても、
　　そのうちフリーズしなくなったっぽい。

  ・256色モードのパレット処理がきれいになったと思う・・・意味ないけど。

　・ウィンドウサイズのフィルタリング付き自由拡大が可能。
　　フィルタ拡大処理がソフトウェアなので、遅い上に暗くなる・・・。
　　ウィンドウのサイズをドラッグして拡大しない限り効果はありません。

　・win9xで使えるかも知れない版同梱。
　　単に古いVCで作成しただけなので動かないかもしれません。
　　win9x以外のOSでも使えます。

　・全体的にちょっと重くなった。
　　負荷が重すぎる人は ddraw_v2 の方で。


現在の機能

　・Windows の全解像度＆全カラーモードに対応。
　　またプレイ中に解像度＆カラーモードの変更が可能。
　・Win7/64bitやVistaのエアロでも動作
　・ウィンドウサイズのフィルタリング付き自由拡大が可能。
　・フルカラー顔表示機能搭載＆簡易アニメ機能付き。


まったく修正する気のない不具合

　・256色モードで起動すると、その他の色モードに移行してもムービーは256色のまま。
　・256色モード以外で起動すると、256色モードに移行したらムービーの色が変。
